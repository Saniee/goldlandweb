Nábory : Popis Požiadavkov/Úloh

Helper

Helper má být jeden z nejaktivnějších členů AT, protože musí pomáhat hráčům což někdy není snadné. Helper taky moderuje chat, obstarává hackery stížnosti, nápady a bugy.


Builder

Builder je zaměřen na stavění různých staveb pro server např. Spawn, mapy. Buildeři musí umět spolupracovat stavět hezky taky i dost rychle, a mít bujnou fantazii.


Manager

Manager je zaměstnán pro to aby dělal reklamu, a slušně propagoval server. Manager je člověk který pomáhá serveru hlavně s hráči.
